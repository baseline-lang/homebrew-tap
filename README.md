# Homebrew Tap

Homebrew formulae for [Baseline](https://github.com/baseline-lang/baseline).

## Installation

```bash
brew install baseline-lang/tap/baseline
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `baseline` | Strongly typed, effect-based programming language |

## Updating

```bash
brew update
brew upgrade baseline
```
